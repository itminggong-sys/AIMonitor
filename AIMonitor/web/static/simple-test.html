🚀 AI监控系统

Windows 安装指南

✅ 测试成功！ 如果您能看到这个页面的样式，说明CSS加载正常。

步骤 1: 系统要求

  • Windows 10 或更高版本
  • Go 1.19 或更高版本
  • Node.js 16 或更高版本
  • 至少 4GB 内存

步骤 2: 下载源码

git clone https://github.com/your-repo/ai-monitor.git cd ai-monitor

步骤 3: 安装依赖

# 安装Go依赖 go mod download # 安装前端依赖 cd web npm install npm run build cd ..

步骤 4: 配置数据库

⚠️ 注意： 请确保已安装并启动 MySQL 数据库服务。
# 复制配置文件 cp config/config.example.yaml config/config.yaml # 编辑配置文件，设置数据库连接信息 notepad config/config.yaml

步骤 5: 启动服务

# 启动后端服务 go run cmd/server/main.go
🎉 服务启动成功！访问 http://localhost:8080 开始使用。
